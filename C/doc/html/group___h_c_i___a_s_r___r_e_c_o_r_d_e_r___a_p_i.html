枚举类型说明

enum RECORDER_ERR_CODE

录音机错误码

枚举值
RECORDER_ERR_UNKNOWN 

-1: 未知错误，不会出现

RECORDER_ERR_NONE 

0: 成功

RECORDER_ERR_NOT_INIT 

1: 没有初始化

RECORDER_ERR_ALREADY_INIT 

2: 已经初始化

RECORDER_ERR_ALREADY_BEGIN 

3: 已经开始录音

RECORDER_ERR_NOT_BEGIN 

4: 没有开始录音

RECORDER_ERR_OUT_OF_MEMORY 

5: 分配空间失败

RECORDER_ERR_ENGINE_ERROR 

6: 调用引擎出错

enum RECORDER_EVENT

录音机回调时的通知事件

枚举值
RECORDER_EVENT_BEGIN_RECORD 

录音开始

RECORDER_EVENT_HAVING_VOICE 

听到声音 检测到始端的时候会触发该事件

RECORDER_EVENT_NO_VOICE_INPUT 

没有听到声音

RECORDER_EVENT_BUFF_FULL 

缓冲区已填满

RECORDER_EVENT_END_RECORD 

录音完毕（自动或手动结束）

RECORDER_EVENT_BEGIN_RECOGNIZE 

开始识别

RECORDER_EVENT_RECOGNIZE_COMPLETE 

识别完毕

RECORDER_EVENT_ENGINE_ERROR 

引擎出错

RECORDER_EVENT_DEVICE_ERROR 

设备出错

RECORDER_EVENT_MALLOC_ERROR 

分配空间失败

RECORDER_EVENT_INTERRUPTED 

iOS收到其他声音中断会触发此事件，同时停止录音机，开发者必须处理此事件

RECORDER_EVENT_PERMISSION_DENIED 

iOS没有麦克风使用权限(ios7以上版本会有此事件，开发者必须处理此事件)

RECORDER_EVENT_TASK_FINISH 

识别任务结束

RECORDER_EVENT_RECOGNIZE_PROCESS 

识别中间状态

enum RECORDER_STATE

录音机状态

枚举值
RECORDER_STATE_NOT_INIT 

没有初始化

RECORDER_STATE_IDLE 

空闲状态，可以开始录音和识别

RECORDER_STATE_RECORDING 

正在录音

RECORDER_STATE_RECOGING 

正在识别

RECORDER_STATE_CONFIRMING 

正在提交用户确认结果

RECORDER_STATE_ERROR 

错误状态，此时录音机已经无法正常工作

RECORDER_STATE_STARTING 

starting session

_OPT_ _IN_ void *  pUsrParam 
)

设置audiosession回调

参数
pfnCallBackaudiosession回调
pUsrParam用户数据
返回

RECORDER_ERR_NONE操作成功
RECORDER_ERR_NOT_INITrecorder尚未初始化

注解
此接口只在iOS平台使用 recorder默认不处理中断。如果你的应用需要响应中断进行相应的处理， 可以调用此函数，设置Callback_RecorderSetAudioSession回调，在回调中进行audiosession的设置。如何设置audiosession，请参 考苹果AudioToolBox的官方文档。下面是一个设置audiosession的示例
void playerSetAudioSessionCallback(void * pExtendParam,void * pUsrParam)
{
//初始化audiosession
AudioSessionInitialize(NULL, NULL, MyAudioSessionInterruptionListener, pUsrParam);
//设置kAudioSessionProperty_AudioCategory
UInt32 sessionCategory = kAudioSessionCategory_PlayAndRecord;
AudioSessionSetProperty ( kAudioSessionProperty_AudioCategory, sizeof (sessionCategory), &sessionCategory);
}
RECORDER_ERR_CODE HCIAPI hci_asr_recorder_start ( _MUST_ _IN_ const char *  pszConfig,
_OPT_ _IN_ const char *  pszGrammarData 
)

录音机SDK启动

配置中包含端点检测配制项

注解
录音机仅提供对 pcm8k16bit pcm16k16bit 格式的支持 如果不是录音机支持的格式，会在 Callback_RecorderEventStateChange 中触发打开设备失败的事件
参数
pszConfig配置串 可通过deviceId指定录音设备，其余参数参见 hci_asr_session_start
pszGrammarData语法文件(只有当配置串grammarType为wordlist、jsgf或wfst时有效）
返回

RECORDER_ERR_NONE操作成功
RECORDER_ERR_NOT_INIT没有初始化
RECORDER_ERR_ALREADY_BEGIN已经开录音
RECORDER_ERR_OUT_OF_MEMORY分配空间失败

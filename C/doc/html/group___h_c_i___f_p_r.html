配置串定义：
配置串是由"字段=值"的形式给出的一个字符串，多个字段之间以','隔开。字段名不分大小写。

字段 取值或示例 缺省值 含义 详细说明
userId abc123 空 用户自定义的唯一标识 用户注册时，用户唯一标识可以自定义；如果为空，就替用户产生一个唯一标识并返回给用户。需要注意，开发者需要确保自定义标识的唯一性。
imageType format,raw format 图像类型 format：格式化图像数据
raw：原始图像编码数据
width 395 无 图像宽度 imageType为format时忽略，raw必选
height 395 无 图像高度 imageType为format时忽略，raw必选
dpi 500 无 每英寸的像素 必选
bgColor white,black 无 图像背景颜色 必选，灰度级0表示黑色，255表示白色
white:图像为白底黑线
black:图像为黑底白线
actionType append,replace replace 注册模型的方式 注册模型有两种方式：
  • replace表示以替换的方式注册模型，如果模型不存在，就新建；如果模型存在，就删除旧模型，建立新模型。
  • append表示以追加的方式注册模型，如果模型不存在，就新建；如果模型存在，就在原来模型的基础上追加训练（暂不支持在原有模型追加）。

HCI_ERR_CODE HCIAPI hci_fpr_init ( _MUST_ _IN_ const char *  pszConfig)

灵云FPR能力 初始化

参数
pszConfig初始化配置串，ASCII字符串，可为NULL或以'\0'结束
返回

HCI_ERR_NONE操作成功
HCI_ERR_SYS_NOT_INITHCI SYS 尚未初始化
HCI_ERR_FPR_ALREADY_INIT已经初始化过了
HCI_ERR_CONFIG_INVALID配置参数有误，如设定值非法、或格式错误等
HCI_ERR_CONFIG_DATAPATH_MISSING缺少必需的dataPath配置项
HCI_ERR_FPR_ENGINE_INIT_FAILED本地引擎初始化失败
HCI_ERR_CONFIG_CAPKEY_NOT_MATCHCAPKEY与当前引擎不匹配
HCI_ERR_CAPKEY_NOT_FOUND没有找到指定的能力
HCI_ERR_LOAD_FUNCTION_FROM_DLL加载库函数失败
HCI_ERR_OUT_OF_MEMORY申请内存失败
HCI_ERR_LOCAL_LIB_MISSING本地能力引擎缺失必要的库资源
配置串定义：
配置串是由"字段=值"的形式给出的一个字符串，多个字段之间以','隔开。字段名不分大小写。

字段 取值或示例 缺省值 含义 详细说明
dataPath opt/myapp/fpr_data/ 空 指纹识别本地资源所在路径 如果不使用本地指纹识别能力，可以不传此项
initCapKeys fpr.local.verify，fpr.local.verify 空 初始化所需的本地能力 多个能力以';'隔开，忽略传入的云端能力key。如果不使用本地识别能力，可以不传此项。
fileFlag none, android_so none 获取本地文件名的特殊标记 参见下面的注释
userModelPath opt/myapp/fpr_data/ 空 用户模型文件存放路径 如果不使用本地指纹识别能力，可以不传此项。使用本地能力，如果用户不设置该项，则使用authpath路径
注解
Android特殊配置
当fileFlag为android_so时，加载本地资源文件时会将正常的库文件名更改为so文件名进行加载。 例如，当使用的库为file.dat时，则实际打开的文件名为libfile.dat.so，这样在Android系统下， 开发者可以按照此规则将本地资源改名后, 放在libs目录下打包入apk。在安装后，这些资源文件 就会放置在/data/data/包名/lib目录下。则可以直接将dataPath配置项指为此目录即可。

配置串定义：
配置串是由"字段=值"的形式给出的一个字符串，多个字段之间以','隔开。字段名不分大小写。

字段 取值或示例 缺省值 含义 详细说明
userId abc123 空 用户唯一标识 此项必填
imageType format,raw format 图像类型 format：格式化图像数据
raw：原始图像编码数据
width 395 无 图像宽度 imageType为format时忽略，raw必选
height 395 无 图像高度 imageType为format时忽略，raw必选
dpi 500 无 每英寸的像素 必选
bgColor white,black 无 图像背景颜色 必选，灰度级0表示黑色，255表示白色
white:图像为白底黑线
black:图像为黑底白线
threshold [0,100] 80 阈值 将图像与指定模型匹配返回的相似百分比，记为相似度。
相似度>=threshold时，返回匹配，否则返回不匹配

char * pTextBuf
 识别结果文本, UTF-8编码, 以'\0'结束，不管配置项 detailResult 设为是否需要详细信息，这项都会存在
 
OCR_RECOG_NODE * pNodes
 识别结果的详细节点列表，只有在配置项 detailResult 设为 yes 的时候有效，否则为NULL
 
int nNodeCount
 识别结果的详细节点的数目，只有在配置项 detailResult 设为 yes 的时候有效，否则为0

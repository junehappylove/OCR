enum CAPTURE_ERR_CODE

*CAPTURE_ERR_CODE 扩展接口错误码定义

枚举值
CAPTURE_ERR_UNKNOWN 

-1: 未知错误，不会出现

CAPTURE_ERR_NONE 

0: 成功

CAPTURE_ERR_ALREADY_INITED 

1: 已经初始化

CAPTURE_ERR_NOT_INITED 

2: 拍图器没有初始化

CAPTURE_ERR_ENGINE_INIT 

3: 引擎初始化失败

CAPTURE_ERR_DEVICE_INIT 

4: 设备初始化错误

CAPTURE_ERR_CAPTURE 

5: 扫描错误

CAPTURE_ERR_LOAD_TEMPLATE 

6: 加载模板错误

CAPTURE_ERR_UNLOAD_TEMPLATE 

7: 卸载模板错误

CAPTURE_ERR_SESSION_START 

8: 启动任务失败

CAPTURE_ERR_RECOGNIZE 

9: 识别失败

CAPTURE_ERR_LOAD_IMAGE 

10: 加载图片失败

CAPTURE_ERR_UNSUPPORT 

11: 接口暂不支持

CAPTURE_ERR_RELEASE 

12: 释放失败

CAPTURE_ERR_BUSY 

13: 状态忙碌，正在拍照或识别

CAPTURE_ERR_CAPTUREONE 

14: 拍照失败

CAPTURE_ERR_PARAM_INVALID 

15: 参数非法

CAPTURE_ERR_OUT_OF_MEMORY 

16: 内存不足

enum CAPTURE_EVENT

*CAPTURE_EVENT 扩展接口回调事件定义

枚举值
CAPTURE_EVENT_CAPTURE_BEGIN 

0: 拍照器开始进行图片捕获

CAPTURE_EVENT_CAPTURE_FINISH 

1: 扫描完成

CAPTURE_EVENT_RECOGNIZE_BEGIN 

2: 开始识别

CAPTURE_EVENT_RECOGNIZE_MORE 

3: 还有图片没有识别

CAPTURE_EVENT_RECOGNIZE_FINISH 

4: 单张图片识别结束

CAPTURE_EVENT_RECOGNIZE_COMPLETE 

5: 所有识别结束

CAPTURE_EVENT_ENGINE_ERROR 

6: 引擎出错，一般为HCI_OCR模块出错

CAPTURE_EVENT_DEVICE_ERROR 

7: 设备出错,目前windows接口都为同步实现，因此都通过返回值返回，不再进行该种错误回调

CAPTURE_EVENT_MALLOC_ERROR 

8: 分配空间失败

CAPTURE_EVENT_PREVIEW_BEGIN 

9: 开始预览图像

CAPTURE_EVENT_PREVIEW_DETECT_CORNER_POINTS_SUCCESS 

10: 检测到预览图片帧的切边信息

CAPTURE_EVENT_PREVIEW_DETECT_CAN_BE_RECOGNIZE_SUCCESS 

11: 检测到预览图片帧可识别

CAPTURE_EVENT_FOCUS_BEGIN 

12: 开始对焦

CAPTURE_EVENT_FOCUS_SUCCESS 

13: 对焦成功

CAPTURE_EVENT_FOCUS_FAIL 

14: 对焦失败

CAPTURE_EVENT_CAPTURE_DETECT_CORNER_POINTS_SUCCESS 

15: 拍照获得的图片检测切边信息成功 暂未使用

CAPTURE_EVENT_CAPTURE_DETECT_CORNER_POINTS_FAIL 

16: 拍照获得的图片检测切边信息失败 暂未使用

CAPTURE_ERR_CODE HCIAPI hci_ocr_capture_recog ( _MUST_ _IN_ const char *  pszConfig,
_MUST_ _IN_ unsigned char *  pImageData,
_MUST_ _IN_ unsigned int  uiImageLen 
)

OCR Capture 支持识别单张图片识别，通过回调返回处理结果

参数
pszConfig识别配置项
pImageData图像数据，可以通过isFile字段来指定使用文件路径还是图像数据内存
uiImageLen设置图像数据的长度，如果接口传入的是路径该设置项无效
配置串定义：
配置串是由"字段=值"的形式给出的一个字符串，多个字段之间以','隔开。字段名不分大小写。并不是每个能力都支持

列表中的所有配置项，各能力支持的配置项请参考OCR能力描述

模块 字段 取值或示例 缺省值 含义 详细说明
识别通用 capKey ocr.cloud 无 OCR识别能力key 参见 OCR能力描述 。每个session只能定义一种能力，并且过程中不能改变。
cutEdge yes, no no 原始图片需要裁边，该配置只适用于本地识别 yes,需要裁边,对于拍照照片适用，原始图像为未裁边图像
no, 不需要裁边，非拍照图片使用，原始图像为裁边图像
isFile yes,no no 是否文件路径输入 是否是文件路径输入
文本识别 recogRange gb, letter all 识别范围 number: 数字 0 - 9, 10个
uppercase: 大写英文字母, A - Z, 26个
lowercase: 小写英文字母, a - z, 26个
letter: 大小写字母, 52个
alnum: 大小写字母和数字, 62个
gb: 中文简体汉字
big5: 中文繁体汉字 all: 所有字符集的合集,英文能力时不包含gb和big5
autoDeskew no, yes no 是否自动倾斜校正，自动识别时才支持
autoLayout no, yes no 是否进行自动版面分析，自动识别时才支持
fullHalf full, half half 设置中文识别模式下输出英文字母、数字是全角还是半角 full: 全角
half: 半角
dispCode nochange, tosimplified nochange 设置输出结果简繁体转换 nochange: 简繁体不做变化
tosimplified: 繁体转换为简体
detailResult no, yes no 是否返回详细结果 缺省只返回识别结果的文本串，如果设为yes，则会返回详细的节点信息 OCR_RECOG_NODE
binarize adaptive, global adaptive 二值化方法，仅对灰度或者彩色图像有效 global: 全局二值化算法，主要针对通过扫描、截屏等获取的图像
adaptive: 局部自适应二值化算法，主要针对通过拍照、摄像头等获取的受光照影响较大图像
imageType normal, screen normal 图像类型 normal: 普通图像
screen: 截屏图像
localProcess yes, no yes 云端能力，设置图像时 是否先做本地预处理，现在主要是指二值化 yes: 二值化
no: 不进行二值化
模板识别 templateID 1 无 模板文件的ID 本地和云端模板识别支持，即capKey为ocr.local.template和ocr.cloud.template时必选，其他情况忽略。
本地识别时，本参数是由 hci_ocr_capture_load_template()返回的ID；
云端识别时，请咨询捷通华声或从灵云开发者社区了解
templateIndex 0到对应模板文件中的最大模板索引号 无 只针对模板识别生效，模板文件中的模板索引号 模板文件中可能有多个模板，比如身份证、票据等，本参数指定模板在模板文件中的索引号
templatePageIndex 0到对应模板中的最大页索引号 无 只针对模板识别生效，模板页索引号 模板中可能存在多个模板页，比如身份证的正面和反面，本参数指定模板页在模板中的索引号
